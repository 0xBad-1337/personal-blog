Bandit Walkthrough

The Bandit wargame is aimed at absolute beginners. It will teach the basics needed to be able to play other wargames. After finishing this levels you will more comfortable with the linux command line, getting resources from the manual pages, googling but more important is getting frustrated for missing the solution :). So let's go
some levels are very easy and obvious . so i will be only showing the interesting ones that i managed to solve

Bandit 6 ==> 7

The password for the next level is stored somewhere on the server and has all of the following properties: owned by user bandit7 / owned by group bandit6 / 33 bytes in size so wee need to find all the files but at the same time we will have a lot of output to deal with so the trick here is to redirect the errors to /dev/null (the blackhole) 

                
                    find / -user bandit7 -group bandit6 -size 33c 2>/dev/null
